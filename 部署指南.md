## 部署步驟

### 方式一：透過 Vercel 網頁（推薦）

1. **登入 Vercel**
   - 前往 [vercel.com](https://vercel.com)
   - 使用 GitHub / GitLab / Bitbucket 登入

2. **匯入專案**
   - 點擊「Add New」→「Project」
   - 選擇「Import Git Repository」
   - 若尚未連接，先將專案推送到 GitHub

3. **設定專案**
   - Framework Preset：**Next.js**（自動偵測）
   - Root Directory：`./`（預設）
   - Build Command：`npm run build`（預設）
   - Output Directory：`.next`（預設）
   - 環境變數：目前無需設定

4. **部署**
   - 點擊「Deploy」
   - 等待建置完成（約 1～2 分鐘）
   - 完成後會得到一個網址，例如：`https://momhealth-xxx.vercel.app`

---

### 方式二：透過 Vercel CLI

```bash

# 2. 在專案目錄執行
cd /Volumes/D/Eric/1-開發/momhealth

# 3. 依提示登入、選擇專案設定
# 4. 正式部署
```

## 注意事項

### 資料儲存
- 目前使用 **localStorage**，資料存在使用者瀏覽器
- 換裝置或清除瀏覽器資料會遺失
- 若未來需要多人同步，可改為 Supabase / Vercel Postgres

### 藥品圖片
- `public/pills/` 與 `public/pills/preview/` 會一併部署
- 請確認圖片已加入 Git 並推送

### 自訂網域（選用）
- 在 Vercel 專案 → Settings → Domains 可綁定自己的網域
