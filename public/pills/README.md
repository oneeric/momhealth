# 藥品照片資料夾

此資料夾放**原圖**，點擊卡片展開時顯示。

另見 `preview/` 資料夾：放**縮圖**，用於卡片上的小圖示。

檔名對應如下（小寫或原廠命名皆可）：

| 檔名範例 | 藥品 |
|----------|------|
| ts1 / TS-1 | TS-1 (愛斯萬) |
| folina / Folina | Folina (芙琳亞) |
| creon / Creon | Creon (卡利消) |
| panzolec / Panzolec | Panzolec (瘍康) |
| emetrol / EmetroL | EmetroL (癒吐寧) |
| traceton / Traceton | Traceton (服安痛) |
| loperam / Loperam | Loperam (肚倍朗) |
| allegra / Allegra | Allegra (艾來) |
| xyzal / XyzaL | XyzaL (驅異樂) |
| tcm | 中藥 |

支援格式：`.jpg`、`.jpeg`、`.png`、`.webp`

若無對應照片，將顯示預設藥丸圖示。
